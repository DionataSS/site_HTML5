HTML5

O que é: HTML5 é a linguagem padrão para estruturar e apresentar conteúdo na web. É a quinta versão do HTML e introduz novos elementos e atributos, proporcionando uma estrutura mais semântica.

O que faz: Define a estrutura básica da página, organiza o conteúdo em cabeçalhos, parágrafos, listas e outros elementos. Também suporta multimídia incorporada, formulários interativos e fornece uma base estrutural para o desenvolvimento web.

CSS

O que é: CSS é uma linguagem de estilo que controla a apresentação e o layout dos elementos HTML em uma página web.

O que faz: Permite a definição de estilos visuais, como cores, fontes, tamanhos, margens e posicionamentos. Ajuda na criação de layouts responsivos e proporciona uma separação clara entre a estrutura HTML e a aparência visual da página.

Javascript

O que é: JavaScript é uma linguagem de programação de alto nível que permite a criação de interatividade dinâmica nas páginas web.

O que faz: Manipula o conteúdo da página em tempo real, responde a eventos do usuário, valida formulários, e interage com o servidor para carregar ou enviar dados. É amplamente utilizado para criar experiências interativas e dinâmicas.
